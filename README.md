# FitLife - Fitness Blogging Website Frontend

A modern, responsive fitness blogging platform built with Next.js, featuring workout guides, nutrition articles, and wellness content.

---

## Description

FitLife is a comprehensive fitness blogging website designed to help users transform their health journey. The platform includes user authentication, workout routines, nutrition guides, and wellness articles, all presented with a clean, modern UI and smooth animations.

##  Features

-  User Authentication (Login / Register)
-  Dynamic Home Page with Hero Section
-  Workout Categories & Guides
-  Nutrition Articles & Meal Plans
-  Wellness & Mental Health Content
-  Fully Responsive Design
-  Smooth Animations & Parallax Effects

---

## Tech Stack

- **Framework:** Next.js 16.0.3  
- **Frontend Library:** React 19.2.0  
- **Styling:** Tailwind CSS 4, CSS Modules  
- **Icons:** FontAwesome  
- **Language:** JavaScript / JSX  

---

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
````

Open the application in your browser:

```
http://localhost:3000
```
